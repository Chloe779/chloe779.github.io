智慧地球=互联网+物联网

物联网的四个阶段：

  1. before2010：基于RFID技术开展局部应用

  2. 2010-2015：制定特定产业的技术标准并完成部分网络的融合

  3. 2015-2020：物体版智能化，物联网中异构网络互联的标准制定完成

  4. after2020：达到人，物，服务与网络的深度融合

普适计算的特点：

  1. 核心以人为本
  2. 体现出信息与物理空间的融合
  3. 强调无处不在与不可见
  4. 提供面向用户的，统一的，自适应的网络服务

CPS:

  1. 一个综合计算，网络与物理世界的复杂系统
  2. 人-机-物深度融合的系统
  3. 3C与物理设备深度融合的系统
  4. 环境感知，嵌入式计算，网络通信深度融合的系统

物联网的特征：

  1. 只能物体具有感知，通信和计算能力
  2. 只能物体是对接到物联网的人和物的一种抽象
  3. 目标是实现物理世界与信息世界的融合
  4. 可以提供所有对象在任何时间，任何地点的互联

物联网和互联网的区别：

  1. 互联网提供面向全球用户的信息交互与共享信息服务
  2. 物联网提供行业性，专业性，区域性的服务
  3. 物联网是可反馈，可控制的闭环系统

物联网体系结构特点：

  1. 物联网分为感知层，网络层与应用层、
  2. 网络层进一步分为接入，汇聚和核心交换曾
  3. 应用层进一步分为管理层与业务应用层

应用层特点：

  1. 应用层分为管理服务层与行业应用层
  2. 管理服务层位于网络层与行业应用层之间
  3. 管理服务层通过中间软件向应用层屏蔽感知层与网络层的差异性
  4. 利用数据挖掘，大数据处理与只能决策技术，为行业应用层提供服务

**信息物理融合系统（CPS)**是环境感知，嵌入式计算，网络通信深度融合的系统，通过3C技术(Computation，Communication and Control)使计算，通信和控制技术邮寄融合、深度写作，实现大型工程系统的试试感知，动态控制和信息服务

物联网只能物体具有感知，通信，计算能力

物联网的目标： 实现物理世界与信息世界的融合

物联网数据主要通过自动方式获取，互联网数据主要是通过人工方式获取

物联网是一个可反馈，可控制的闭环系统

感知层设备主要分为两类： 自动感知设备和人工生成信息设备

物联网数据特点：还良性，多态性，动态性，关联性

物联网关键技术：

感知技术，计算技术，通信与网络技术，嵌入式技术，智能技术，位置服务技术，网络安全技术，物联网应用系统规划与设计技术

物联网的共性服务

  1. 信息安全
  2. 网络管理
  3. 对象名字服务
  4. 服务质量保证

简述物联网三层结构模型：

物联网分为感知层，网络层和应用层

  1. 感知层是物联网的基础，包含各种感知层设备，入自动感知设备与人工生成信息设备
  2. 网络层主要有核心交换曾，汇聚层， 接入层。接入层通过各种接入技术，链接最终用户设备；汇聚层聚合接入层的用户流量，实现数据路由，转发与交换；核心交换曾为物联网提供一个高速，安全与保证服务质量的数据传输环境
  3. 应用层分为管理服务层与行业应用层。管理服务层用过中间软件对应用层软件屏蔽了感知层的感知设备以及网络层传输网络的差异性，将还想感知数据汇聚存储，利用数据挖掘，大数据处理与智能决策技术，为行业应用层提供服务

CPS主要功能：

  1. 感：多感知器协同感知物理世界的状态信息
  2. 联：链接物理世界与信息世界的各种对象，实现信息交互
  3. 知：通过对感知信息的智能处理，正确，全面地认知物理世界
  4. 控：根据正确的认知，确定控制策略，发出指令，只会执行器协同控制物理世界

RFID

RFID是利用无线射频信号空间耦合的方式，实现无接触的标签信息自动传输与识别的技术，又称电子标签或射频标签

RFID标签读写器的基本功能：

对固定或移动RFID标签进行识别与读写，发现读写过程中出现的错误

将读取的RFID存储的数据传送到计算机，将计算机写入的数据或指令发送到RFID芯片

简述RFID读写器的结构：

  1. 中心控制器模块：对RFID读写器整体运行进行控制
  2. 读写器模块：对标签的数据读出与写入
  3. 存储器模块：存储系统软件，应用软件，标签数据
  4. 人机交互模块：实现操作人员命令，显示执行结果
  5. 接口模块：实现读写器与高层计算机的数据通信
  6. 电源模块：监控手持设备的电源供应与电池电量

传感器Sensor是由敏感元件和转换元件组成的一种检测装置

生物传感器室友 生物铭感原件 和 信号传感器 组成

无线传感器网络节点类型：

  1. 传感器节点
  2. 汇聚节点
  3. 管理结点

WSN

无线传感器网络 Wireless Sensor Networks,是由部署在检测区域内大量的，廉价的卫星传感器节点组成，通过无线通信方式形成的一个多跳的，自组织的无线自组网系统，其目的是将网络覆盖区域内感知对象的信息发送给观察者

传感器，感知对象，**观察者 ** 构成无线传感器网络的三个要素

特点：

  1. 异构型
  2. 实时性
  3. 协作性
  4. 移动性

无线传感器网络特点：

  1. 网络规模与应用需求相关
  2. 自组织网络：传感器节点具有自组织能力，能够自动进行配置和管理，通过拓扑控制机制和网络协议，自动形成转发检测数据的多跳无线网络系统
  3. 拓扑结构的动态变化
  4. 以数据为中心

无线传感器节点结构：

  1. 传感器模块：信息的感知和采集，将模拟信号转化成数字信号
  2. 处理器模块：控制整个传感器节点的操作，存储和处理传感器采集的数据，以及其他节点传来的数据
  3. 无线通信模块：与其他传感器节点进行无线通信，网络层模块负责选择数据包的传输路由，戒指访问控制模块负责协调多节点对公共通信信道的访问控制，无线收发器模块负责数据信号的发送和接受
  4. 能量供应模块：由微型电池与电源控制电路组成，提供运行所需要的能量

GIS:
